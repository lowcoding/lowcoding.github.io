lowcode

低代码工具

提效工具

支持抽离常用代片段、功能区块为模板（物料），代替频繁复制粘贴。

适用范围广

与项目使用的框架、库甚至语言无关，能在重复的代码中抽离出模板即可。

可视化操作

可直接通过 webview 页面进行操作。

物料开发、共享

支持自定义物料开发，并且通过 git 或者 npm 包的方式在不同项目中共享。
